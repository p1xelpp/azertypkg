# vpkg
this is the vpkg repo
vpkg is just an ordinary package manager, BUT because it depends like..... everything, BUT i made the dependencies, so they are available un bundled :D

*WARNING*: only available on posix systems (idk if i actually WANT to make a win port)

**setup**

this is the easy way to install vpkg

1. make "cp2bin" executable
2. run sudo ./cp2bin cp2bin to cp2bin cp2bin (it means ur copying cp2bin to bin (cp2bin is a program to put things in /usr/local/bin))
3. cp2bin rmbin, bin2home, vpkg-install and vpkg (optionally, you can also cp2bin vellpkg-build)
4. run vpkg config
5. add a package repo with vpkg add-repo <url> (you can make ur own repo btw, just look at https://github.com/p1xelpp/vpkg-public, i am working on making it working, if you ACTUALLYwanna see whats in the 1st package, download it and run vpkg-install <i forgor the name of the package so package name here>
and thats it
